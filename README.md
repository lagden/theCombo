# jquery.theCombo.js

jQuery plugin for **customize** the `<select>` tag.

    See example!!!

## Installation

Install with [Bower](http://bower.io):

```
bower install --save jquery.theCombo.js
```

The component can be used as a Common JS module, an AMD module, or a global.
